# ComicMaster 素材市场 — 开发需求与对接文档

## 定位

独立站点，展示个人制作的漫画素材包。与 ComicMaster 本地工具零耦合，仅通过一个外链对接。

**定位**：免费展示 + 免费下载。付费交易完全在第三方平台完成，本站不处理任何支付。

## 站点结构

```
market.yourdomain.com
├── /                    首页（精选素材包展示）
├── /packs               素材包列表（筛选/搜索）
├── /pack/<slug>         素材包详情页
├── /download/<code>     购买后下载（带下载码）
└── /about               关于 / 使用说明
```

## 页面需求

### 首页
- Hero 区：一句话说明"ComicMaster 官方素材市场"
- 精选素材包卡片（3-6 个，手动置顶）
- 底部说明：如何在 ComicMaster 中使用

### 素材包列表
- 卡片网格布局，每个卡片：封面图、名称、价格、标签数、文件大小
- 筛选：按角色 / 风格 / 类型
- 排序：最新 / 最热

### 素材包详情
- 封面大图
- 名称、价格、简介
- 包含标签列表（预览前 10 个，其余折叠）
- 预览图（4-8 张素材截图）
- 文件大小、图片数量、分辨率
- 购买按钮 → 跳转支付
- 购买后显示下载按钮 / 下载码输入框

### 下载页
- 输入下载码 → 校验 → 返回文件下载
- 下载码一次性使用或限时有效

## 数据模型

```
material_pack:
  id          int
  slug        str       # URL 友好标识
  name        str       # 素材包名称
  description str       # 简介（Markdown）
  price       int       # 分（微信支付单位）或 元（支付宝单位）
  cover_url   str       # 封面图
  previews    [str]     # 预览图列表
  tags        [str]     # 包含的标签列表（用于展示和搜索）
  file_count  int       # 图片数量
  file_size   str       # 文件大小（如 "156MB"）
  resolution  str       # 分辨率（如 "2000x2000px"）
  category    str       # 分类：角色/场景/道具/特效/分镜模板
  file_path   str       # 服务器上的 zip 文件路径
  is_featured bool      # 首页精选
  is_active   bool      # 上架/下架
  created_at  datetime

order:
  id           int
  pack_id      int
  download_code str      # 唯一随机码
  amount       int       # 实付金额
  status       str       # paid / downloaded / expired
  paid_at      datetime
  downloaded_at datetime
```

## 法规与合规

### 个人站红线

| 事项 | 要求 |
|------|------|
| ICP 备案 | 国内服务器必须做（个人可办，免费） |
| ICP 经营许可证 | 个人无法办理，自建支付/付费下载/会员系统属无证经营 |
| 收款 | 个人不能私挂收款码或私域转账 |
| 版权 | 所有素材包必须原创或已获授权 |

### 本站合规方案

- **本站**：免费展示 + 免费下载，不内置任何支付功能
- **付费引导**：仅放跳转按钮，指向第三方平台
- **交易**：在第三方平台完成（收款、发货、售后）
- **内容**：免费版引流，付费版仅展示功能描述

### 支付对接：面包多（mbd.pub）

个人实名即可开店，无需营业执照，无需 ICP 许可：

- 支持微信支付 + 支付宝
- 数字商品自动发下载链接
- 手续费 5%

### 对接流程

1. 面包多后台创建商品，填写名称、价格、上传素材包 zip
2. 商品发布后获得支付链接
3. 素材市场站点中，购买按钮跳转面包多链接
4. 用户支付后，面包多自动发送下载链接
5. 用户下载 zip → 解压到 ComicMaster 的 `input/words/` 目录

### 补充：其他合规第三方平台

除面包多外，以下平台也适用于个人创作者：
- **微信小商店**：微信官方，个人可开，支持自动发货
- **爱发电**：创作者订阅/单品购买
- **各类发卡平台**：数字商品自动发货

切换支付平台只需修改 `packs.js` 中的 `pay_link` 字段，前端无变化。

## 技术选型

### 推荐方案：静态站点 + 少量后端

| 层 | 技术 | 原因 |
|----|------|------|
| 前端 | 纯 HTML/CSS/JS 或 Hugo/Next.js 静态生成 | 素材包内容不常变，静态够用 |
| 后端 | Python Flask（最小化） | 只需要下载码校验 + 文件分发 |
| 存储 | 服务器本地 / 阿里云 OSS | 素材包 zip 文件 |
| 部署 | 阿里云轻量服务器 / Vercel + 云函数 | 低成本，国内访问快 |
| 域名 | 阿里云/腾讯云注册 | .com 或 .cn |

### 最简方案

如果完全是面包多支付：

- **零后端**：纯静态 HTML 站点
- 展示页面用 Hugo / Astro 生成
- 购买按钮跳面包多链接
- 无需服务器、无需数据库、无需支付代码
- 部署到 GitHub Pages / Vercel / 阿里云 OSS 静态托管

## ComicMaster 对接

只需在 ComicMaster 侧边栏添加一个外链按钮：

```html
<!-- webapp/templates/index.html 侧边栏底部 -->
<a class="sidebar-settings-link" href="https://market.yourdomain.com" target="_blank" title="素材市场">
  <span class="sidebar-settings-icon">🛒</span>
  <span>素材市场</span>
</a>
```

就这么简单。不涉及 API 调用、不涉及认证、不涉及文件传输。用户自己去商店买，买完自己解压到本地素材目录。

### 后续可选增强

- 下载码输入框：ComicMaster 设置页加一个输入框 + 按钮，输入下载码后自动拉取 zip 并解压
- 需要商店后端增加一个 `/api/download/<code>` 接口（约 30 行 Python）
- ComicMaster 侧增加下载脚本（约 20 行 Python）

## 素材包制作规范

### 目录结构

```
素材包名称.zip
├── 角色名/
│   ├── 表情1.png
│   ├── 表情2.png
│   ├── 动态1.png
│   └── ...
├── 场景名/
│   └── ...
└── README.txt（可选：标签说明、使用建议）
```

### 图片规范

| 参数 | 建议值 |
|------|--------|
| 格式 | PNG（透明背景优先） |
| 分辨率 | 2000px 以上短边 |
| 命名 | 中文描述性名称，如 `微笑正面.png` |
| 色彩 | RGB 或 RGBA |

### 标签命名建议

- 与 ComicMaster 素材库标签体系统一
- 多级标签用 `/` 分隔，如 `角色/主角/微笑`
- 标签名应描述视觉内容而非文件名

### 数据包产品定价与分层策略
#### 核心思路
- 高阶抽象分镜数据独家且小众，需**免费基础版教育市场**，降低感知门槛
- 用免费版建立认知，再通过付费版承接转化，合规且差异化

#### 产品分层
##### 1. 免费基础版（引流）
- 极简泛化分镜规律、基础镜头/分格数据
- 让用户直观感知工具价值，吸引精准用户

##### 2. 轻量付费版（大众进阶）
- 定价：**29.9～39.9 元**
- 完整场景分镜结构、条漫/页漫节奏模型

#### 3. 高阶专业版（独家壁垒）
- 定价：**89～129 元**
- 全维度大数据、齐夫定律分布模型、行业级结构化数据

#### 视觉词汇小包（走量）
- 单品小包：**9.9 元**
- 系列合集：**29.9～39.9 元**

关于素材市场

本市场由 ComicMaster 作者独立运营，提供高品质漫画素材包。所有素材均按 ComicMaster 标签体系制作，确保与工具的完美兼容。

使用方式

  1. 浏览并选择需要的素材包
  2. 在详情页扫码支付（微信 / 支付宝），支付后获取百度网盘链接和提取码
  3. 通过网盘下载素材包 zip 文件
  4. 将 zip 解压到 ComicMaster 素材目录：input/words/
  5. 在 ComicMaster 中刷新标签索引，即刻可用

素材标准

  • PNG 格式，透明背景（便于叠加和组合）
  • 短边分辨率不低于 2000px
  • 与 ComicMaster 标签体系完全一致，解压即命中
  • 中文描述性命名，直观可检索

支付说明

本站为个人创作站点，所有交易均为个人直接交易。付费素材包通过微信 / 支付宝个人收款码完成支付，支付后即可获取百度网盘下载链接。

请妥善保存支付成功后的微信/支付宝交易记录，作为购买凭据。如忘记提取码，可凭交易记录截图联系客服找回。

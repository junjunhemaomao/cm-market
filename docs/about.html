关于素材市场

本市场由 ComicMaster 作者独立运营，提供高品质漫画素材包。所有素材均按 ComicMaster 标签体系制作，确保与工具的完美兼容。

使用方式

  1. 浏览并选择需要的素材包
  2. 通过面包多支付（支持微信 / 支付宝），支付后获取下载码
  3. 在本站输入下载码，获取 zip 文件
  4. 将 zip 解压到 ComicMaster 素材目录：input/words/
  5. 在 ComicMaster 中刷新标签索引，即刻可用

素材标准

  • PNG 格式，透明背景（便于叠加和组合）
  • 短边分辨率不低于 2000px
  • 与 ComicMaster 标签体系完全一致，解压即命中
  • 中文描述性命名，直观可检索

图片更新与部署

所有素材包的封面图和预览图存放在 public/images/ 目录下。
你可以用本地图片编辑工具（如 Photoshop、GIMP、SAI）直接修改这些图片，或放入自己的真实截图。
运行 python generate_examples.py 可重新生成示例占位图。
完成后部署整个 public/ 目录到服务器或静态托管平台即可。

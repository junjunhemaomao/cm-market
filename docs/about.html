关于素材库

本素材库由 ComicMaster 作者独立运营，提供高品质漫画素材包。所有素材均按 ComicMaster 标签体系制作，确保与工具的完美兼容。

使用方式

  1. 浏览并选择需要的素材包
  2. 在详情页获取百度网盘链接和提取码，直接下载
  3. 将 zip 解压到 ComicMaster 素材目录：input/words/
  4. 在 ComicMaster 中刷新标签索引，即刻可用

素材标准

  • PNG 格式，透明背景（便于叠加和组合）
  • 短边分辨率不低于 2000px
  • 与 ComicMaster 标签体系完全一致，解压即命中
  • 中文描述性命名，直观可检索
